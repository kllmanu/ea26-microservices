# Exercise 4

Extend your existing e-commerce microservice project from Exercises 1–4 with
observability, and analyse the architecture of the Spring Petclinic
Microservices reference system.

## Requirements:

- Add application metrics, at least two domain metrics, to your services (at least two services) and make them available (Micrometer/Actuator)
- Collect the metrics on a central server (e.g. Prometheus)
- Create a dashboard (e.g. Grafana) and display the metrics
